**Modules:**

Meta - Utility Commands

Mod - Moderation Commands

Config - Logging Commands

Misc - Miscellaneous Commands

Games - Game Commands

Music - Music Commands

&nbsp;

```
Syntax for Sirk:
<input> - Input for the Command
[o] - Optional
"|" - Alias
(permission) - Permission
```
